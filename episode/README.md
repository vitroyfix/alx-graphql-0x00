# Episode Queries

This folder contains GraphQL queries to fetch episodes from the Rick and Morty API.

- `episode-id-1.graphql` → Query episode with ID 1
- `episode-id-2.graphql` → Query episode with ID 2
- `episode-id-3.graphql` → Query episode with ID 3
- `episode-id-4.graphql` → Query episode with ID 4
