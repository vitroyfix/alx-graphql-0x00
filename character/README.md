# Character Queries

This folder contains GraphQL queries to fetch characters from the Rick and Morty API.

- `character-id-1.graphql` → Query character with ID 1
- `character-id-2.graphql` → Query character with ID 2
- `character-id-3.graphql` → Query character with ID 3
- `character-id-4.graphql` → Query character with ID 4
- `characters-page-1.graphql` → Query characters list, page 1
- `characters-page-2.graphql` → Query characters list, page 2
- `characters-page-3.graphql` → Query characters list, page 3
- `characters-page-4.graphql` → Query characters list, page 4
